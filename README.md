## 🎯 About The Project

This is my personal portfolio website showcasing my journey as a **Full Stack Developer**. Built with React and Tailwind CSS, it highlights my skills, projects, certifications, internships, and achievements.

### Why this portfolio?
- ✅ Clean, modern, and responsive design
- ✅ Dark/Light mode toggle for better user experience
- ✅ Permanent resume upload feature (saved in browser's localStorage)
- ✅ Fully customizable without backend
- ✅ Mobile-first approach

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🌓 **Dark/Light Mode** | Toggle between themes with persistent storage |
| 📄 **Resume Upload** | Upload PDF/DOC resume - saved permanently in browser |
| 🏆 **Achievements Showcase** | Display hackathon wins and academic achievements |
| 💼 **Internships Timeline** | Current and completed internships with descriptions |
| 📜 **Certifications Grid** | NPTEL, IIT, and other certifications with badges |
| 🚀 **Projects Gallery** | Featured projects with tech stack tags |
| 📧 **Contact Form** | Stores messages in localStorage (no backend needed) |
| 📱 **Fully Responsive** | Works on all devices |
